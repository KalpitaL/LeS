Quotations

Quotations are to be submitted within fourty eigth hours after the query has been received from KGJS. Should you however require more time to process the query then please send an email to purchase@kgjs.no
When submitting a quote please send to purchase@kgjs.no, and ensure at all times that the vessel name and query number is included in the subject box. The following information is to be included on quotations:

  • Payment: Net per 30 days
  • Delivery-terms: As per Incoterms 2010
  • Vessel Name
  • KGJS reference number
  • Your reference number
  • Your contact person
  • Please ensure your item numbers correspond with the KGJS query sequence
  • Unit price / Total price
  • Currency
  • Discount rate
  • Delivery time
  • Place from where delivery will/can be done
  • Shipping/handling/packaging costs if applicable. This is to be noted separately from the product price
  • Please note that KGJS have their own freight insurance arrangements and any additional charges by the Vendor will not be accepted
  • Validity of the offer
  • Terms of payment
  • Approximate weight/dimensions if available at the time of quoting

When submitting quotes for technical spares it should be clearly stated whether your company is offering original or none original parts.
To enable the KGJS Purchasing department to process the queries in a more efficient manner we require all written correspondence to be in the English language.
